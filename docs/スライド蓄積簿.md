# スライド蓄積簿

公開スライドカタログ（index.html）の元データ。1 行 = 1 スライド。カタログは scripts/build-catalog.mjs がこの表から再生成する。

## 記入規則

- スライドキーは連番禁止。内容を要約した意味語で付ける（番号からは情報を得られないため）
- 1スライド = 1フォルダ（`slides/<スライドキー>/解説スライド.html`）。フォルダごと配布できる状態を保つ
- 各フォルダに `検査記録.md` を残す（観点レビュー表の照合結果と、そのスライド限定の観点。更新時の再検査項目になる）
- 公開リポジトリのため、特定環境の絶対パス・特定顧客の情報はスライドにも本表にも含めない
- 行を追加・変更したら `node scripts/build-catalog.mjs` を実行して index.html を再生成する
