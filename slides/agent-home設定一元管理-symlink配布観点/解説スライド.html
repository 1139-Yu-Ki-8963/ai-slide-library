環境構築

AI エージェントの設定資産は 1 リポジトリに集約し、
シンボリックリンクで各ツールに配布する

設定の分散を排除し、変更の一貫性を構造で保証する方式

正本（git リポジトリ）
~/agent-home/
全ツール共通の設定資産を一元管理
rules/ 36 本 行動規約
skills/ 14 本 手順書
agents/ 10 体 サブエージェント定義
routines/ 14 本 定期タスク
tools/   共有ツール・linter
templates/   プロジェクト雛形
配布先（シンボリックリンク）
Claude Code 接続中
~/.claude/rules → agent-home/rules/
~/.claude/skills → agent-home/skills/
~/.claude/agents → agent-home/agents/
Codex CLI 接続中
~/.codex/rules → agent-home/rules/
将来のツール 拡張可能
リンク 1 本で同じ設定を即時共有
1
正本が 1 箇所 設定の分散・乖離が構造的に不可能
2
git で変更管理 履歴追跡・差分レビュー・ロールバック
3
リンク 1 本で即配布 新ツール追加も即座に同じ設定を共有
