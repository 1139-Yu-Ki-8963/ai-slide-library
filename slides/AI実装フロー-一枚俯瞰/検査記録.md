## 反復記録

1. 初回描画: 全カードが `.flow-row { flex:1 }` によりスライド残り高さ全体まで伸長し、カード内テキストの下に大きな余白が生じた（内容自体の欠落・切れはなし。バランスの問題）
2. 修正: `.flow-node` に `justify-content: center` を追加して中央寄せを試行 → カード自体が依然として過大な高さのままで、視覚的な間延び感は解消せず
3. 再修正: `.flow-row` を `flex: 0 0 auto; margin: auto 0;` に変更し、カード列を内容に応じた自然な高さにしたうえでスライド残り空間内で縦中央配置 → 余白バランスが解消（3回目の再描画で確認、上限3回以内）
