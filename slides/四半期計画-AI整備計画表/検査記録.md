# 検査記録: 四半期計画-AI整備計画表

## 観点レビュー

| 観点 | 結果 | 根拠 |
|---|---|---|
| タイトルが主張文 | PASS | 「AI整備計画表（4段階構成）」で対象と構成が明示 |
| 結論の単一性 | PASS | 「年間計画を四半期軸で俯瞰できる」の1主張 |
| 段階の期間表示 | PASS | 4段階を矢印バーで期間表示。開始・終了四半期ラベル付き |
| 段階の重なりの明示 | PASS | 1Q後半〜2Qと2Q〜3Qが矢印バー上で視覚的に重複 |
| 四半期軸の一貫性 | PASS | 1Q〜4Qの4分割グリッドを全行で共有 |
| 自己完結性 | PASS | 外部依存なし。CSS のみでインライン描画（矢印バーは clip-path） |
| ダークモード対応 | PASS | prefers-color-scheme で配色反転、data-theme 明示指定にも対応 |
| 16:9 比率 | PASS | 1280×720px |
