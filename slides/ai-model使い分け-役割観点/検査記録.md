## タスク限定の観点（更新時に再検査する）

- Cursor の auto モードは品質に問題がある（設計書を最後まで読まない等）。推奨しない立場を維持する
- リミット到達時の代替は Cursor の Composer モデルに限定する
- Claude Code の3モデル（Opus/Sonnet/Haiku）の役割分担は一般的な使い分け指針として記述し、特定PCの構成（サブエージェント数等）を含めない
- ツール識別マーク: Claude Code = コーラル放射状マーク、Cursor = チャコール立方体マーク
- モデル識別色: Opus = 紫（--opus）、Sonnet = コーラル（--claude）、Haiku = 青（--haiku）
- 推奨/非推奨バッジ: 推奨 = 緑（--good）、非推奨 = 赤（--critical）
