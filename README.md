# ai-slide-library

AIツールの論点を横1枚（16:9）で説明する解説スライドHTMLの公開ライブラリ。

- **カタログ**: https://1139-yu-ki-8963.github.io/ai-slide-library/
- 悩み・キーワードで検索し、その場で全画面提示、または1クリックでダウンロードできます

## 使い方

- **見る**: カタログのカード（サムネイルまたは「提示する」）を押すと、新しいタブで全画面表示されます
- **落とす**: 「ダウンロード」を押すと単体のHTMLファイルとして保存されます。スライドは1ファイルで完結しており、ブラウザで開くだけで表示できます（インターネット接続不要）
- カタログ自体もローカルで動きます。リポジトリを clone して index.html を開くだけで、検索・フィルタを含む全機能が使えます

## スライドの追加手順（管理者向け）

1. `slides/<スライドキー>/解説スライド.html` を作成する（スライドキーは連番禁止。内容を要約した意味語で付ける）
2. `docs/スライド蓄積簿.md` の「スライド一覧」表に1行追記する
3. `node scripts/build-catalog.mjs` を実行して index.html を再生成する
4. commit して push すると GitHub Pages に自動反映される

## 構成

```
ai-slide-library/
├── index.html            カタログ（検索・観点フィルタ・提示・ダウンロード）
├── slides/
│   └── <スライドキー>/
│       ├── 解説スライド.html   スライド本体（1280×720・自己完結）
│       └── 検査記録.md         観点レビューの照合記録
├── scripts/
│   └── build-catalog.mjs  カタログ生成スクリプト（Node.js・依存なし）
└── docs/
    └── スライド蓄積簿.md    スライドのメタデータの正本
```
